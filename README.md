# 🚇 Bihar Metro Route Optimization System

An intelligent metro route and fare optimization system built using **Python**, **Dijkstra's Algorithm**, **MongoDB**, and **Google Maps API**.

This project helps commuters find the **shortest route** and **accurate fare calculation** between metro stations in Bihar.

---

## 📌 Problem Statement

Existing metro systems often provide:
- Static route information
- No intelligent optimization
- Fare confusion among passengers
- Lack of real-time integration

This project solves these issues using graph algorithms and live data.

---

##  Features

-  Shortest path calculation using Dijkstra’s Algorithm
-  Dynamic fare calculation
-  Real-time integration with Google Maps API
-  User-friendly GUI built with Tkinter
-  Graph-based metro network visualization
-  MongoDB database integration

---

## 🛠 Tech Stack

- Python
- NetworkX
- MongoDB
- Tkinter
- Google Maps API
- Dijkstra's Algorithm
